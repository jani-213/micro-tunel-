proseso para hacer el microtunel

Un microtúnel es un túnel pequeño que se utiliza para instalar cables, tuberías o otros servicios subterráneos sin necesidad de excavar zanjas profundas. A continuación, se presenta el proceso general para hacer un microtúnel:

1. Planificación y diseño

- Se realiza un estudio de factibilidad para determinar la viabilidad del proyecto. - Se diseñan los planos y especificaciones del microtúnel, incluyendo la ruta, el diámetro y la profundidad.

2. Preparación del sitio

- Se prepara el sitio de trabajo, incluyendo la limpieza y la señalización del área. - Se instalan los equipos de seguridad necesarios.

3. Perforación

- Se utiliza una máquina de perforación horizontal (HDD) para crear un pilote guía. - Se avanza gradualmente, utilizando una broca y un sistema de guiado para mantener la dirección y la profundidad.

4. Instalación del microtúnel

- Una vez que se ha completado la perforación, se instala el microtúnel. - Se puede utilizar una variedad de materiales, como tuberías de PVC o acero.

5. Instalación de servicios

- Se instalan los servicios deseados, como cables o tuberías, dentro del microtúnel.

6. Sellado y protección

- Se sella el microtúnel para evitar la entrada de agua o suciedad. - Se protege el microtúnel con materiales adecuados para evitar daños.

Ventajas

- Menor impacto ambiental en comparación con la excavación tradicional. - Reducción de costos y tiempos de construcción. - Mayor seguridad para los trabajadores y el público.

Aplicaciones

- Instalación de cables de fibra óptica o eléctricos. Instalación de tuberías de agua, gas o alcantarillado. - Cruce de carreteras o vías férreas sin interrumpir el tráfico.
